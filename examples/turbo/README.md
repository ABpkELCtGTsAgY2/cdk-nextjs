# Turbo Example

./extensions folder was downloaded from S3. S3 URL was generated by running: `aws lambda get-layer-version-by-arn --arn $LAYER_ARN --query 'Content.Location' --output text`. Look up your ARN [here](https://docs.aws.amazon.com/systems-manager/latest/userguide/ps-integration-lambda-extensions.html#ps-integration-lambda-extensions-add). Pay attention to x86 or ARM64.
